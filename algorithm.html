Решение алгоритмической задачи

Условие задачи №1011

Определить значение функции Z при произвольных X и Y по формуле:

Z = 1 / (X × Y)

где X и Y - произвольные числа, отличные от нуля.

Введите данные для расчета

Результат вычислений

Введите значения X и Y и нажмите "Вычислить Z"

Алгоритм решения

  1. Получить значения переменных X и Y от пользователя
  2. Проверить, что X ≠ 0 и Y ≠ 0
  3. Вычислить произведение P = X × Y
  4. Вычислить значение функции Z = 1 / P
  5. Вывести результат пользователю

Математическая запись:

Z(X, Y) = 1 ÷ (X · Y)

Область определения:

Функция определена для всех X ∈ ℝ и Y ∈ ℝ, таких что X ≠ 0 и Y ≠ 0.
